
## create jest-config

- `pnpm exec jest --init`

## types

add `@types/jest` to tsconfig types

## paths

`paths` in tsconfig should same as `moduleNameMapper` in jest.config.js

## testMatch mach test files

```json
  testMatch: ["**/__tests__/**/*.[jt]s?(x)", "**/?(*.)+(spec|test).[tj]s?(x)"],
```
